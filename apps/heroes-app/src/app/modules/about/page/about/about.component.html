Description of the project

Github

The project is made of 3 pages, heroes, villains and about. Heroes has the grid and the crud. From this page we can see and filter heroes and delete. We can navigate to "create" page to edit or create a hero. It's also a PWA so it can be installed. It has a bit of everything (directives, pipes, animations, providers, store, reactive streams, rest services and local storage)

Heroes are fetched from this api but is read only so to edit create and delete we combine this info with a local database made with PouchDB. Heroes created,edited or deleted are stored in this database and merged with the data from the api. This may cause some mismatch between the pagination number from the api and the final number of elements.

Packages used

  • Taiga ui
  • Akita
  • PouchDb
  • Ngx-translate
